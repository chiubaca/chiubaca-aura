## Installation

### From Source (Local)

1. Copy this folder to:
   - **macOS**: `~/.vscode/extensions/chiubaca-aura`
   - **Windows**: `%USERPROFILE%\.vscode\extensions\chiubaca-aura`
   - **Linux**: `~/.vscode/extensions/chiubaca-aura`

2. Restart VS Code

3. Go to `Preferences > Color Theme` and select **Chiubaca Aura**

### From VSIX (Package)

```bash
# If you have vsce installed
vsce package

# Install the generated .vsix file
code --install-extension chiubaca-aura-1.0.0.vsix
```
